CUSTOMIZED FIRMWARE

To install the customized firmware you need to copy the "templ_chip.txt"
from the "compile" folder inside this ZIP file to one of the following
folders on your computer:

Windows

  C:\Users\YourName\AppData\Local\Mindflood Ltd\Patchblocks\compile
  
You need to replace YourName with your actual Windows username.

Mac

  /Applications/Patchblocks.app/Contents/MacOS/compile

Note that this will overwrite the original template, so you might want to
backup the original before proceeding.
